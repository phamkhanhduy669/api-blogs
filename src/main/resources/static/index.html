Chào mừng tới Blog App

Ứng dụng demo Spring Boot + JWT. Bạn có thể đăng ký user, đăng nhập để lấy token, và thao tác với blog cá nhân. ADMIN có thể xem tất cả và xoá user.

Sử dụng Postman gọi /api/auth/login vẫn trả về JSON chứa token như bình thường.
